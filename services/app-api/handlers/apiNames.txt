-- coresets
GET /coreset/{state}/{year} - gets all coresets tied to state and year
GET /coreset/{state}/{year}/{coresetID} - gets coreset with matching ID -- might not need this
POST /coreset/{state}/{year}/{coresetID} - creates coreset associated with state and year (req.body unknown)
PUT /coreset/{state}/{year}/{coresetID} - edits coreset associated with state and year (req.body unknown)
DELETE /coreset/{state}/{year}/{coresetID} - deletes coreset associated with state year and id 

-- measures
GET /coreset/{state}/{year}/{coresetID}/measures - gets all measures tied to state and year
GET /coreset/{state}/{year}/{coresetID}/measures/metadata - get measures metadata (id,status,name,abbrev) with specific id
GET /coreset/{state}/{year}/{coresetID}/measures/{measureId} - get specific measure (returns all data for that meausre)
POST /coreset/{state}/{year}/{coresetID}/measures/{measureId} - creates measure with specific measure id
PUT /coreset/{state}/{year}/{coresetID}/measures/{measureId} - edit specific measure information

DELETE /coreset/{state}/{year}/{coresetID}/measures/{measureId} - delete specific measure
